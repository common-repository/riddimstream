RiddimStream plugin for WordPress

== Description ==

This plugin allows you to embed any music Riddimstream on your website or blog . It works for any RiddimStream track. Once installed, you can click the Riddimstream button in any of your posts or pages to insert a shortcode.

A shortcode example:

`[riddimstream id="{ID}" width="600" height="200"]`

**More Options**

These parameters are optional and default to the recommended height

* `width`
* `height`


== Installation ==

1. Upload `riddimstream` to your plug-in directory or install it from the Wordpress Plug-in Repository
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit a post and click the Riddimstream button. 
4. Paste the URL to the track you wish to play - Riddimstream will automatically find the correct ID for the shortcode.
